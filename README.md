# pnfl-profilewriter

Updates a Front Page Sports Football Pro '98 coaching profile (.prf).

**Not yet implemented.** Skeleton only; the analog of [`pnfl-gameplanwriter`](../pnfl-gameplanwriter) for `.prf` files.

When built out, it will sit on top of [`pnfl-profile`](../pnfl-profile) (which wraps [`fbpro98-profile`](../fbpro98-profile) with PNFL rule data) and expose a `pnfl write-profile` umbrella subcommand.
